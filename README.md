# ⚽ GoPlay

Plataforma completa para **gestão de quadras society**, campeonatos e controle financeiro.

O GoPlay foi criado para donos de quadras, organizadores de campeonatos e gestores esportivos controlarem reservas, times, campeonatos e recebimentos em um único sistema.

---

## 🚀 Tecnologias

### Backend
- Node.js
- Express
- Prisma ORM
- PostgreSQL / MySQL
- JWT

### Frontend
- HTML
- CSS
- JavaScript
- Bootstrap
- SB Admin 2

### Infraestrutura
- Linux Server
- Nginx
- PM2
- HTTPS

---

## 📦 Funcionalidades

### 🏟️ Societies (Quadras)
- Cadastro de quadras
- Gestão de horários
- Controle de reservas
- Dados da quadra

### 🏆 Campeonatos
- Criação de campeonatos
- Times participantes
- Tabela e classificação
- Gestão de partidas

### 💰 Recebimentos
- Controle de pagamentos
- Caixa por society
- Histórico financeiro
- Relatórios

### 👤 Usuários
- Login e autenticação
- Perfil do gestor
- Controle de acesso

### 📊 Dashboard
- Visão geral da operação
- Indicadores financeiros
- Status das quadras
- Próximos jogos
